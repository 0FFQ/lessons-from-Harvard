Команды для Git

  • git clone <url> - клонирует репозиторий из интернета на локальный компьютер.
  • git add - добавляет файлы, изменения в которых нужно отслеживать и сохранять в следующем коммите.
  • git add . - сохраняет все файлы; git add style.css index.html - только определённые файлы.
  • git commit - сохраняет текущее состояние файлов в репозитории (делает «снимок»), можно использовать с параметром -m для указания сообщения к коммиту.
  • git commit -am "сообщение" - сохраняет все файлы (как git add .) и сразу делает коммит.
  • git push - загружает изменения с локального компьютера на удалённый сервер (например, на GitHub).
  • git pull - скачивает последние изменения с удалённого сервера на локальный компьютер.
  • git status - показывает текущее состояние репозитория: какие файлы изменены, какие подготовлены к коммиту и т. д.
  • git log - выводит список всех коммитов с информацией о них (хэш, автор, дата, сообщение).
  • git reset - возвращает репозиторий к предыдущему состоянию, можно использовать с разными параметрами, например --hard для жёсткого сброса.
  • git branch - показывает существующие ветки и текущую ветку, на которой находится пользователь.
  • git checkout - переключает между ветками или создаёт новую ветку с параметром -b.
  • git merge - объединяет изменения из одной ветки в другую.
  • ni style.css - создаёт CSS-файл как New-Item.
  • cd "путь" - зайти в папку в терминале; cd .. - вернуться на папку назад.
